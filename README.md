# Traveling Salesman Problem (TSP) Solver

TSP Solver implemented in Python.

## Algorithms
1. Hill Climbing.
2. Random Restart Hill Climbing.
3. Tabu Search.

## Requirements
* Python 3.10 or higher.
* tsplib95.
* matplotlib.

## How to Use
```bash
pip install -r requirements.txt
python3 main.py instances/ar24.tsp
